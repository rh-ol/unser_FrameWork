# Willkommen beim Robusten Automatisierungs-Framework (V3.11)

**Willkommen, Entwickler!**

Dieses Ökosystem (V3.11) ist der Standard zur Erstellung von #robusten, #wartbaren und #sicheren Automatisierungs-Workern.

Der Einstieg wird vollständig von einem **KI-Partner (V3.11)** gesteuert.

---

### Der "One Shot" Workflow (Ihr Start)

1.  **Starten Sie einen neuen Chat** mit dem KI-Partner.
2.  Kopieren Sie den **gesamten Inhalt** des `KI_BOOTSTRAP_PROMPT.md (V3.11)` und fügen Sie ihn als erste Nachricht ein.

Die KI wird Sie ab diesem Moment #proaktiv führen:
1.  Sie wird Sie bitten, Ihr #Zielsystem zu **analysieren** (P0) und stellt Ihnen das Werkzeug (`system_profiler.sh`) dafür bereit.
2.  Nach der Analyse wird sie Sie nach dem **Projekttyp** fragen (P0):
    * **[1] Greenfield (Neu)**
    * **[2] Wrapper (Stabilisierung)**
    * **[3] Rewrite (Ablösung)**
